BeRocket Custom Thank You Page is a extendable WP plugin that helps you better thank your customer after order.

== Description ==

Custom Thank You Page is necessary solution that gives store owners opportunity to thank customers for their orders.

= General thank you page =
Choose 1 page that will be shown after all customer orders.

= Show order details  =
Hide or Show order details.

= Show cross-sell products  =
Hide or Show cross-sell products that are connected with ordered product.

= Show upsell products =
Hide or show upsell products that are connected with ordered product.

= Thank you page for separate product  =
Choose thank you page for separate product.

= Intuitive interface  =
Understandable and comfortable access to all settings.

== Installation ==

= Minimum Requirements =

* PHP version 5.2.4 or greater (PHP 5.6 or greater is recommended)
* MySQL version 5.0 or greater (MySQL 5.6 or greater is recommended)+
